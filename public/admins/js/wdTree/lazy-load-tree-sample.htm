Lazy Load Nodes This demo show how load nodes in a lazy way.
If you have a lots of levels/nodes(more 10,000 nodes) in a tree and load it in one time, you may concern lazy load way. That is, the highest level nodes are loaded initially. Subnodes will be loaded by ajax call only if end user would like to expand their parent node. Nodes not expands will never be loaded. You can expand/collapse and check/uncheked nodes to see how quickly it response.
Refresh Node The following lines show how to get checked nodes. 
          $("#reflashnode10").click(function(e) {
                $("#tree").reflash("root-0"); //"root-0" is id of the node to be reloaded
            });
